# DesafioAlura2
Analisis de tiendas
##Propósito del análisis

El propósito del análisis es evaluar el desempeño de cuatro tiendas utilizando datos de ventas con el fin de identificar cuál presenta el menor rendimiento financiero y, con base en esa información, recomendar una decisión estratégica para el negocio. A través del análisis de indicadores como ingresos totales, volumen de ventas, categorías de productos más vendidas y costos de envío, se busca comprender el comportamiento comercial de cada tienda. Este estudio permite apoyar la toma de decisiones basada en datos, especialmente en la determinación de qué tienda conviene vender o cerrar para optimizar los recursos de la empresa y mejorar su rentabilidad general.

Estructura del proyecto y organización de los archivos

El proyecto se encuentra organizado en un notebook de Python, donde se realiza todo el proceso de análisis de datos utilizando la biblioteca pandas para la manipulación de datos y matplotlib o herramientas similares para la visualización.
La estructura general del proyecto es la siguiente:
El notebook contiene las siguientes secciones principales:
Importación de librerías
Se cargan las librerías necesarias para el análisis de datos.
Carga de datos
Se importan los archivos CSV de las cuatro tiendas desde los enlaces proporcionados.
Análisis de facturación
Se calcula el ingreso total de cada tienda sumando la columna de precios de los productos vendidos.
Análisis de ventas por categoría
Se identifican las categorías de productos más vendidas.
Análisis de productos más vendidos
Se determinan los productos con mayor volumen de ventas.
Análisis de costos de envío
Se compara el costo promedio de envío entre las tiendas.
Visualización de resultados
Se generan gráficos para facilitar la interpretación de los datos.

Ejemplos de gráficos e insights obtenidos

Durante el análisis se generaron diversos gráficos que permiten visualizar el desempeño de cada tienda.
Gráfico de ingresos totales por tienda
Se creó un gráfico de barras que muestra los ingresos generados por cada tienda. Este gráfico permite identificar rápidamente cuál tienda tiene mayor y menor rendimiento económico.
Gráfico de categorías más vendidas
Se visualizaron las categorías de productos con mayor número de ventas. Este análisis ayuda a entender qué tipo de productos tienen mayor demanda en cada tienda.
Gráfico de costos promedio de envío
Se compararon los costos de envío entre las tiendas para evaluar su impacto en la rentabilidad.
